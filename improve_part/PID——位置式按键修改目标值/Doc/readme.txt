【*】 程序简介 
-工程名称：PID算法实现\位置式按键修改目标值
-实验平台: 野火STM32 F103 指南者开发板
-MDK版本：5.30
-HAL库版本：V1.1.4

【 ！】功能简介：
通过按下KEY1以设置PID的目标值,按下KEY2以使能\失能PID计算.

【 ！】实验操作：

通过按下KEY1以设置PID的目标值,按下KEY2以使能\失能PID计算.

打开野火多功能试助手----PID调试助手,在调试助手中打开开发板对应串口,上电开发板,按下KEY1\KEY2即可观察PID调整过程.

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

将开发板串口与电脑连接.
